## ⚡ Quick Start (2 Minutes)

### 1️⃣ Open in Android Studio
```
File → Open → Select this folder (PCSensei-Android)
```

### 2️⃣ Wait for Gradle Sync
- Takes 1-2 minutes first time
- Shows progress in bottom bar

### 3️⃣ Start Backend Server
**In a new terminal:**
```bash
cd D:\Project Work\PCSensei
python -m http.server 8000
```
Leave it running.

### 4️⃣ Click Run ▶️
- Select device/emulator
- App builds and launches!

**Done!** 🎉

## 🎯 What This App Does

**PCSensei** is an AI-powered PC building assistant:

✨ **Desktop PC Builder**
- Enter budget (₹15k to ₹5L+)
- Select usage (Gaming, Productivity, etc.)
- Get complete build with 7 components

✨ **Laptop Finder**
- AI-powered recommendations
- Multi-factor scoring
- Top 3 picks for your budget

✨ **Smart Features**
- Budget allocation optimization
- Socket compatibility checking
- PSU wattage calculation
- Shop links to 6 retailers
- Price formatting (₹)
